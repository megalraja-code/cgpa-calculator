# CGPA Pro Max

A web-based CGPA and SGPA calculator with prediction and analytics features.  
Designed to help students track academic performance and plan future semesters.

## Features
- Real-time CGPA calculation
- SGPA prediction based on target CGPA
- What-if analysis for grade planning
- Subject-wise performance visualization (Chart.js)
- Local storage for data persistence
- Add/remove subjects dynamically
